# acrostic

acrostic.

![img](acrostic)


## Requirements

- norns

## Documentation

- E1 changes page
- K1 shifts

### chord selection

- E2 select chord position
- E3 change chord
- K2 start/stop
- K3 transpose chords

### chord manipulation

- E2 select chord notes
- E3 rotate notes
- K2/K3 lower/raise octave
- K1+K3 reset octaves

### sequence arrangement

- E2 select sequence notes
- E3 rotate notes
- K2/K3 lower/raise octave
- K1+K3 reset octaves

### sampling

- E2 select sample
- E3 change level
- K2/K3 queue recording
- K1+K2 erase recording
- K1+K3 queue all recordings
